# MSN Messenger XML importer
> A tool to read your old MSN Messenger XML chat logs

## Installation
This is a Vue.js project built using `vue-cli`. To work on this project clone the repo, then:

    npm install

And use

    npm serve

For development and

    npm build

To build the project
